# kimono-lesson-fee-calculator
# 授業料・教材費計算レジアプリ (Kimono Lesson Fee Calculator)

着付け教室の業務効率化（DX）を目的として開発した、iOSネイティブ風デザインのWebアプリケーションです。
Google Apps Script (GAS) とスプレッドシートをデータベースとして活用し、顧客種別ごとの複雑な割引ロジックや、マスターデータの自己修復機能を実装しています。

## セキュリティに関する注意事項
セキュリティおよび個人情報保護の観点から、公開しているコードは以下の通り加工しています。
* スプレッドシートの接続URL（ID）はプレースホルダーに変更済
* アプリのログインパスワードはダミー文字列に変更済
* 本番環境で稼働している実在の講師データ・個人情報はすべて除外（空データ）設定済

ご自身の環境でテスト利用される場合は、`Code.gs` の `SHEET_URL` と、`index.html` の `MY_PASS` をご自身の環境に合わせて設定してください。
